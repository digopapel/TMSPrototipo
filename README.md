# NexaTMS Prototype

NexaTMS e um prototipo de TMS moderno para transportadoras, criado para validar uma experiencia mais simples, visual e produtiva para operacao de cargas.

O projeto usa fluxos comuns do transporte rodoviario, como coletas, viagens, tracking e baixa de entrega, mas com uma interface propria e mais atual do que sistemas legados.

## Objetivo

Criar a base de um sistema de gestao de transportes com foco em:

- Usabilidade para operadores, motoristas e gestores.
- Visao rapida da operacao diaria.
- Rastreamento claro para clientes.
- Menos telas complexas e mais fluxo guiado.
- Evolucao futura para backend, banco de dados, mobile e integracoes fiscais.

## Como abrir

Abra `index.html` no navegador.

O app e estatico e nao precisa de servidor, instalacao ou dependencias.

## Estrutura do projeto

- `index.html`: estrutura das telas e modulos.
- `styles.css`: layout, responsividade e identidade visual.
- `app.js`: dados simulados e interacoes do prototipo.
- `ROADMAP.md`: plano de evolucao do produto.

## Modulos atuais

- Dashboard operacional com KPIs, fluxo por status e prioridades.
- Coletas com formulario de programacao e fila filtravel.
- Viagens e manifestos com progresso, documentos e paradas.
- Tracking de remessa com consulta simulada e linha do tempo.
- Baixa de entrega com fluxo mobile para motorista e comprovante digital.

## Status

Este repositorio esta na fase de prototipo funcional.

Os dados ainda sao simulados no frontend. A proxima etapa natural e transformar estes fluxos em uma aplicacao real com backend, banco de dados e autenticacao.
